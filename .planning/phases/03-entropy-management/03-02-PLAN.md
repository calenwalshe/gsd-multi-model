Create test fixtures with temp git repos. Use `make_fixture` helper. For age testing, create a file with a TODO, commit it, then use `GIT_AUTHOR_DATE` and `GIT_COMMITTER_DATE` environment variables to backdate the commit (e.g., 45 days ago) so git blame reports a known date.

Test structure:
```bash
# Test: finds TODOs
make_fixture "finds-todos"
echo '# TODO: fix this' > "$FIXTURE_DIR/src.sh"
git add -A && git commit -q -m "add todo"
OUTPUT=$(bash "$CHECK_SCRIPT" --project-root "$FIXTURE_DIR" 2>/dev/null)
# Assert: findings array contains the TODO

# Test: age computation with backdated commit
make_fixture "age-tracking"
PAST_DATE="2026-01-01T00:00:00Z"
echo '# TODO: old item' > "$FIXTURE_DIR/old.sh"
git add -A
GIT_AUTHOR_DATE="$PAST_DATE" GIT_COMMITTER_DATE="$PAST_DATE" git commit -q -m "old commit"
OUTPUT=$(bash "$CHECK_SCRIPT" --project-root "$FIXTURE_DIR" 2>/dev/null)
# Assert: age_days > 60 (since we backdated to Jan 1)

# Test: untracked file
make_fixture "untracked"
echo '# TODO: new item' > "$FIXTURE_DIR/untracked.sh"
# Don't git add
OUTPUT=$(bash "$CHECK_SCRIPT" --project-root "$FIXTURE_DIR" 2>/dev/null)
# Assert: age_days = 0
```

**Then write `bin/check-stale-todos.sh`**:

1. Parse arguments: `--project-root <path>`, `--warn-days N`, `--critical-days N`. Defaults: project root = parent of script dir, warn = 30, critical = 90.

2. Read thresholds from `.planning/config.json` if present (using node -e pattern). CLI args override config values.

3. Find TODO/FIXME: `grep -rn "TODO\|FIXME"` across source files (`.sh`, `.js`, `.cjs`, `.ts`, `.md`), excluding `.git/`, `node_modules/`, `.planning/`.

4. For each match, extract file and line number, then:
   - Check if file is git-tracked: `git ls-files --error-unmatch "$file" 2>/dev/null`
   - If tracked: `git blame -p "$file" -L "$line,$line"` and parse `author-time` field
   - If untracked: use current timestamp
   - Compute age: `age_days = (now_epoch - author_time) / 86400`

5. Classify severity: `info` (age < warn_days), `warning` (warn <= age < critical), `critical` (age >= critical_days).

6. Output JSON to stdout, human summary to stderr. Exit 0 if no findings, exit 1 if any findings.

Make both scripts executable.
