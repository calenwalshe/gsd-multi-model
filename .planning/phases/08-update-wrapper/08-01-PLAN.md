<objective>
Create `bin/gsd-update.sh` -- a single-command update wrapper that chains GSD framework update, addon reinstall, and compatibility verification with structured exit codes. Users can update GSD and reinstall the addon in one step instead of manually running two commands and forgetting the reinstall.

Output: New `bin/gsd-update.sh` script with three-stage pipeline and `bin/test-gsd-update.sh` test suite.
</objective>

Create `bin/gsd-update.sh` following the prescriptive structure from 08-RESEARCH.md.

**Script structure:**

1. **Header block** (shebang, set -euo pipefail, header comment documenting purpose and exit codes 0/1/2/3)

2. **SCRIPT_DIR and REPO_ROOT resolution:**
```bash
SCRIPT_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")" && pwd)"
REPO_ROOT="$SCRIPT_DIR/.."
```

3. **ANSI color helpers** (install.sh style, TTY detection on fd 1, stdout output):
- `ok()`, `warn()`, `err()` with checkmark/warning/cross characters
- Match install.sh pattern, NOT codex-task.sh pattern (this is a user-facing script)

4. **Inline semver_compare()** -- identical implementation to install.sh, copy verbatim:
- Split on `.` using `local IFS=.`
- Compare major/minor/patch as integers
- Return via echo: -1/0/1

5. **Pre-flight check:**
- Verify `$REPO_ROOT/install.sh` exists, exit 1 if not found

6. **Record old GSD version** (for comparison reporting per 08-RESEARCH.md Open Question 1):
```bash
VERSION_FILE="$HOME/.claude/get-shit-done/VERSION"
OLD_VERSION=""
[ -f "$VERSION_FILE" ] && OLD_VERSION=$(cat "$VERSION_FILE" | tr -d '[:space:]')
```

7. **Banner:**
```
===========================================================
 gsd-update -- Update GSD + Reinstall Addon
===========================================================
```
Use `=` box characters (matching install.sh pattern).

8. **Stage 1: GSD Update**
- Print `==> Stage 1: Updating GSD framework...`
- Run `npx -y get-shit-done-cc@latest --all --global || GSD_RC=$?`
- Do NOT pipe through tee (preserve TTY detection for GSD installer)
- If GSD_RC != 0: `err "GSD update failed (exit code: $GSD_RC)"`, exit 1
- If GSD_RC == 0: `ok "GSD framework updated"`

9. **Stage 2: Addon Reinstall**
- Print `==> Stage 2: Reinstalling addon...`
- Run `bash "$REPO_ROOT/install.sh" --force || INSTALL_RC=$?`
- Let install.sh output flow through directly (user benefits from seeing full log)
- If INSTALL_RC != 0: `err "Addon reinstall failed (exit code: $INSTALL_RC)"`, exit 2
- If INSTALL_RC == 0: `ok "Addon reinstalled"`

10. **Stage 3: Version Report + Compatibility Verification**
- Print `==> Stage 3: Verifying compatibility...`
- Read VERSION file, report new version with `ok "GSD version: v${NEW_VERSION}"`
- If OLD_VERSION is set and differs from NEW_VERSION, report transition: `ok "Updated: v${OLD_VERSION} -> v${NEW_VERSION}"`
- If VERSION file missing: `warn "VERSION file not found after update"`, exit 0
- If python3 unavailable or gsd-compat.json missing: `warn` and exit 0
- Read min/max from gsd-compat.json using python3
- Run semver_compare to check if NEW_VERSION is within range
- If in range: `ok "GSD v${NEW_VERSION} is within tested range (${min} - ${max})"`
- If out of range: `warn "GSD v${NEW_VERSION} is outside tested range (${min} - ${max})"`, print closing banner with warning note, exit 3
- If VERSION format invalid: `warn "GSD VERSION has invalid format: $NEW_VERSION"`, exit 0

11. **Success summary banner:**
```
===========================================================
 UPDATE COMPLETE
 GSD: v${NEW_VERSION}
===========================================================
```
Exit 0.

**Key implementation rules:**
- Initialize `GSD_RC=0` before npx call, use `|| GSD_RC=$?` (prevents set -e abort)
- Initialize `INSTALL_RC=0` before install.sh call, use `|| INSTALL_RC=$?`
- Guard all python3 calls with `command -v python3 &>/dev/null`
- Guard python3 JSON reads with `|| { warn "..."; exit 0; }` fallback
- Validate VERSION format with `[[ "$NEW_VERSION" =~ ^[0-9]+\.[0-9]+\.[0-9]+$ ]]`
- Make the file executable: `chmod +x bin/gsd-update.sh`

  # Copy gsd-compat.json

  MOCK_HOME=$(mktemp -d)
  mkdir -p "$MOCK_HOME/.claude/get-shit-done"

create_mock_npx() {
  local exit_code="$1"
  cat > "$MOCK_DIR/bin/npx" << EOF
#!/bin/bash
echo "Mock GSD install (exit $exit_code)"

EOF
  chmod +x "$MOCK_DIR/bin/npx"

create_mock_install() {

EOF
  chmod +x "$MOCK_DIR/install.sh"
}

cleanup_mock_env() {
  rm -rf "$MOCK_DIR" "$MOCK_HOME"
}
