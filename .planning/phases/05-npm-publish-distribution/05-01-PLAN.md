2. In bin/cli.sh, add a bash semver comparison function near the top (after variable declarations):

```bash
version_gte() {
  local IFS=.
  local i ver1=($1) ver2=($2)
  for ((i=0; i<${#ver2[@]}; i++)); do
    if ((${ver1[i]:-0} < ${ver2[i]:-0})); then return 1; fi
    if ((${ver1[i]:-0} > ${ver2[i]:-0})); then return 0; fi
  done
  return 0
}
```

3. After the existing GSD base detection (where cli.sh checks if GSD is installed), add version compat checking:
