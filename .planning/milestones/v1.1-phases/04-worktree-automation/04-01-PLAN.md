  <name>Task 1: Create bin/worktree-create.sh</name>
  <files>bin/worktree-create.sh</files>
  <behavior>
    - No args: creates worktree at ../gsd-worktree-{repo}-{short-hash} on branch gsd/worktree/{short-hash}
    - --task path/to/04-01-PLAN.md: derives branch gsd/phase-04/plan-01 and dir ../gsd-worktree-phase04-plan01
    - --json: outputs JSON summary instead of human-readable text
    - Dirty working tree: exits 1 with "clean your working tree" message
    - Branch already exists: exits 2 with "branch already exists" message
    - Not a git repo: exits 1 with "not a git repository" message
    - Existing worktree at target path: exits 2 with "worktree already exists" message
    - 3+ active GSD worktrees: prints warning but proceeds
  </behavior>
  <action>
    Create `bin/worktree-create.sh` with the following structure. Follow the shell style from `install.sh` (set -euo pipefail, ANSI color helpers with TTY detection, ok/warn/err functions).

    **CLI interface:**
    - `bin/worktree-create.sh` — auto-generate branch/dir names
    - `bin/worktree-create.sh --task path/to/PLAN.md` — derive names from plan file
    - `bin/worktree-create.sh --json` — machine-readable output
    - `bin/worktree-create.sh --base <commit>` — create from non-HEAD commit (Claude's discretion)
    - Combined flags supported: `--task PLAN.md --json`

    **Branch naming (per user decisions):**
    - With `--task path/to/04-01-PLAN.md`: parse phase number and plan number from filename, generate `gsd/phase-04/plan-01`
    - Without `--task`: generate `gsd/worktree/{8-char-git-short-hash}-{4-char-random-suffix}` for collision avoidance
    - Collision avoidance: if branch exists, append `-2`, `-3`, etc. (up to 10 attempts, then fail)

    **Worktree directory naming (per user decisions — sibling dirs):**
    - With `--task`: `../gsd-worktree-phase{NN}-plan{NN}` (relative to repo root)
    - Without `--task`: `../gsd-worktree-{8-char-hash}-{suffix}`

    **Pre-flight checks (all must pass before creating):**
    1. Verify inside a git repository (`git rev-parse --git-dir`)
    2. Verify clean working tree (`git diff --quiet && git diff --cached --quiet`), exit 1 if dirty
    3. Verify branch name not taken (`git show-ref --verify`), exit 2 if exists
    4. Verify no existing worktree at target path, exit 2 if exists
    5. Count active GSD worktrees (`git worktree list` + grep for gsd-worktree), warn if >= 3

    **Exit codes (per user decisions):**
    - 0: success
    - 1: dirty tree, not a git repo, or general pre-flight failure
    - 2: branch exists or worktree path exists

    **Success output (human-readable):**
    ```
    ═══════════════════════════════════════════════════════
     GSD Worktree Created
    ═══════════════════════════════════════════════════════
      Branch:    gsd/phase-04/plan-01
      Path:      /absolute/path/to/gsd-worktree-phase04-plan01
      Base:      abc1234 (HEAD)
      Command:   cd /absolute/path/to/gsd-worktree-phase04-plan01
    ═══════════════════════════════════════════════════════
    ```

    **Success output (--json):**
    ```json
    {"branch":"gsd/phase-04/plan-01","path":"/abs/path","base_commit":"abc1234","base_ref":"HEAD"}
    ```

    Make the script executable (`chmod +x`).
  </action>

    ```
      gsd/phase-04/plan-02     /path/to/gsd-worktree-phase04-plan02   2m ago
    ```

    6. If no GSD worktrees found, print "No active GSD worktrees."
    7. `--json` outputs array of objects: `[{"branch":"...","path":"...","commit":"...","age_seconds":300}]`
