  <name>Task 1: Create bin/worktree-cleanup.sh</name>
  <files>bin/worktree-cleanup.sh</files>
  <behavior>
    - bin/worktree-cleanup.sh gsd/phase-04/plan-01: merges branch, removes worktree dir, deletes branch, prints merge summary
    - bin/worktree-cleanup.sh gsd/phase-04/plan-01 --json: same but JSON output
    - bin/worktree-cleanup.sh --no-merge --force gsd/phase-04/plan-01: removes worktree and branch without merging
    - bin/worktree-cleanup.sh --all: finds all GSD worktrees, merges each sequentially, stops on first conflict
    - bin/worktree-cleanup.sh --all --no-merge --force: discards all GSD worktrees
    - Dirty main working tree: exits 1 with "clean your working tree" message
    - Merge conflict: exits 3, runs git merge --abort, lists conflicting files
    - Branch not found: exits 1 with "branch not found" message
    - No branch argument (without --all): exits 1 with usage message
  </behavior>
  <action>
    Create `bin/worktree-cleanup.sh` following the same shell conventions as `bin/worktree-create.sh` (set -euo pipefail, ANSI colors with TTY detection, ok/warn/err helpers).

    **CLI interface:**
    - `bin/worktree-cleanup.sh <branch>` — merge and remove single worktree
    - `bin/worktree-cleanup.sh --no-merge --force <branch>` — discard without merging
    - `bin/worktree-cleanup.sh --all` — merge all GSD worktrees sequentially
    - `bin/worktree-cleanup.sh --all --no-merge --force` — discard all
    - `bin/worktree-cleanup.sh <branch> --json` — machine-readable output
    - Combined flags supported in any order

    **Single worktree cleanup flow (per user decisions):**
    1. Parse arguments: extract branch name, flags (--no-merge, --force, --json, --all)
    2. Pre-flight: verify git repo, verify clean working tree in main repo (exit 1 if dirty)
    3. Resolve worktree: find worktree path for given branch via `git worktree list --porcelain`
    4. If branch not found in worktree list, exit 1

    **If --no-merge --force:**
    5a. Run `git worktree remove --force <path>`
    6a. Run `git branch -D <branch>`
    7a. Print "Worktree discarded" message

    **If normal merge (default):**
    5b. Run `git merge <branch> --no-ff` (merge commit preserves history per user decision)
    6b. If merge fails (conflict):
        - Capture list of conflicting files from `git diff --name-only --diff-filter=U`
        - Run `git merge --abort`
        - Print conflicting file list to stderr
        - Exit 3
    7b. On successful merge, capture merge stats:
        - Files changed, insertions, deletions from `git diff --stat HEAD~1`
        - Merge commit hash from `git rev-parse HEAD`
    8b. Run `git worktree remove <path>`
    9b. Run `git branch -d <branch>` (safe delete — branch is merged)

    **Success output (human-readable):**
    ```
    ═══════════════════════════════════════════════════════
     GSD Worktree Merged
    ═══════════════════════════════════════════════════════
      Branch:      gsd/phase-04/plan-01
      Merge commit: abc1234
      Files changed: 3
      Insertions:   +42
      Deletions:    -7
    ═══════════════════════════════════════════════════════
    ```

    **Success output (--json):**
    ```json
    {"branch":"gsd/phase-04/plan-01","merge_commit":"abc1234","files_changed":3,"insertions":42,"deletions":7}
    ```

    **--all mode:**
    - List all GSD worktrees by calling `bin/worktree-list.sh --json` (parse the JSON array to get branch names)
    - For each worktree, run the single cleanup flow
    - Stop on first failure (conflict or error), report which worktree failed
    - Print summary at end: "Cleaned N of M worktrees"

    **Exit codes (per user decisions):**
    - 0: success
    - 1: dirty tree, not a git repo, branch not found, missing args
    - 3: merge conflict (after aborting merge and listing conflicting files)

    Make the script executable (`chmod +x`).
  </action>
